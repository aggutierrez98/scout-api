## Project Quick Reference

- **Stack**: Node.js v22 / TypeScript / Express / Prisma 7 / Turso / Redis / Infisical
- **API prefix**: `/api`
- **All routes** require `checkSession` **except** `/api/webhook/*` (uses `webhookAuth`)
- **IDs**: always `uuid` externally, never expose integer `id`
- **Roles**: `EXTERNO < COLABORADOR < EDUCADOR < JEFE_RAMA < ADMINISTRADOR`
- **Permissions format**: `{accion}_{recurso}` (e.g. `create_pago`, `read_scout`)
- **Secrets**: via `SecretsManager.getInstance()` — never `process.env` for sensitive values
- **Errors**: always `AppError` — never raw `Error`
